# Unit System:
Mass: $kg$ \
Length: $mm \space (10^{-3}m)$ \
Acceleration: $mm/s^2 \space (10^{-3} m/s^2)$ \
Force: $kg \cdot mm/s^2 \space (10^{-3} N)$ \
Stress: $\frac{Force}{length^{2}} = \frac{10^{-3} N}{10^{-6}m^2} = 10^3 Pa$
